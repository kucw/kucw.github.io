LeetCode 刷題筆記 - 217. Contains Duplicate

古古

2024/10/30

題目 #

LeetCode 連結

給定一個 array nums，如果其中有任何一個 value 出現過至少兩次，就 return true，否則 return false

Example 1

Input: nums = [1,2,3,1]
Output: true

Example 2

Input: nums = [1,2,3,4]
Output: false

Example 3

Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true

解答 #

程式實作 #

  • 秒殺題，用 Hash Set 就可以解決這個問題，時間複雜度為 ${O(n)}$ ，並且空間複雜度為 ${O(n)}$ （因為額外使用到了一組 Hash Set 的長度）

    • 唯一要注意的細節大概就是 Python 中的 Hash Set 的用法….初始化 Set 是用 set() 方法，並且加入一個元素是用 add()！

      class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        num_set = set()

        for num in nums:
            if num in num_set:
                return True
            else:
                num_set.add(num)

        return False

時間複雜度、空間複雜度分析 #

  • 時間複雜度： ${O(n)}$
  • 空間複雜度： ${O(n)}$
