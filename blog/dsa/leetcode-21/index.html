LeetCode 刷題筆記 - 21. Merge Two Sorted Lists

古古

2024/10/30

題目 #

LeetCode 連結

給定兩個已經被排序過的 linked list list1 和 list2，合併這兩個 linked list 成一個有序的 linked list，並且返回合併過後的 linked list 的 head

Example 1

Input: list1 = [1,2,4], list2 = [1,3,4]
Output: [1,1,2,3,4,4]

Example 2

Input: list1 = [], list2 = []
Output: []

Example 3

Input: list1 = [], list2 = [0]
Output: [0]

解答 #

程式實作 #

  • 這題非常簡單，但還是有兩個小亮點可以學習，也就是 「dummy head」 和一個 linked list 的小優化
  • 這題解法的思路，其實就是用一個 while loop，不斷的去比較 list1 和 list2 目前的值誰小，誰小就先選哪個，直到兩個 list 都指到 None 為止，就表示 loop 完畢，非常簡單

    • 不過在實作時，一開始可以創建一個 dummy head，可以用來簡化 edge case 的判斷

    • 第二個小亮點，是當有一個 list 已經指到 None 時，就直接把後面的 list2 接上，然後就可以 break 了，不用跑完後續所有 node

      # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:

        # 第一個亮點：使用 dummy_head 簡化第一個 node 的判斷，後續的 node 就全部接在 dummy_head 後面就好
        dummy_head = ListNode()
        current = dummy_head

        while list1 or list2:

            # 第二個亮點，當 list1 已經走到 None 時，就直接把後面的 list2 接上，然後就可以 break 了，不用跑完後續所有 node
            if list1 is None:
                current.next = list2
                break

            elif list2 is None:
                current.next = list1
                break

            elif list1.val <= list2.val:
                current.next = list1
                current = current.next
                list1 = list1.next
            else:
                current.next = list2
                current = current.next
                list2 = list2.next

        # 最後回傳 dummy_head.next，從這裡開始才是正確的返回值
        return dummy_head.next

時間複雜度、空間複雜度分析 #

  • 時間複雜度： $O(n + k)$ ，其中 n 為 list1 的節點個數、k 為 list2 的節點個數
