LeetCode 刷題筆記 - 206. Reverse Linked List

古古

2024/10/30

題目 #

LeetCode 連結

給定一個單向的 linked list 的 head，反轉這個 linked list，然後返回反轉過後的 linked list 的 head

Example 1

Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]

Example 2

Input: head = [1,2]
Output: [2,1]

Example 3

Input: head = []
Output: []

解答 #

程式實作 #

  • 解答的關鍵在於 「使用兩個 pointer，prev 和 current」，這樣子就可以實作出反轉的 linked list

    • 這題解答要先用畫圖來看，畫完圖之後就能輕鬆的實作了

    • 首先這是我們擁有的 input

    • 然後我們想要把他轉變成這個 output

    • 所以我們的目標，就是用兩個 pointer prev 和 current，去實作這個目的

  • 一開始先把 prev 設定成 null，然後 current 設定成 head node（也就是 1）

    • 然後把 current 的 next 設成 prev，這樣子就完成 1 這個 node 的反轉了

    • 所以後面就可以依樣畫葫蘆，先把 current 指到下一個 node（也就是 2），然後把 prev 指到目前 current 的 node，接著再一樣改一下 next 的值，就可以完成整條 list 的反轉了

    • 不過在這裡要特別注意一下，因為這個改 next 的過程中，會丟失原本的 next 資訊，所以就會使得 current 沒辦法從 1 走到 2（因為 1 的 next 已經被改成指到 null 了，所以在改 next 之前，要先把 1 的 next 記錄一下，這個是實作上的細節要注意）

    • 而執行到最後之後，這時候剛好 prev 的 pointer 會指到最後一個 node（也就是 4）上面，因此到最後只要把 head 改成 prev 的值，就可以取得到新的反轉 list 的 head 了

  • 程式實作

    • 基於上面的圖的運作邏輯，可以寫出下列的程式

      # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        prev, current = None, head

        while current is not None:
            next_node = current.next  # 把 current.next 先記錄一下
            current.next = prev

            prev = current
            current = next_node

        # 新的 head 的值就是 prev
        new_head = prev
        return new_head

  • 這一題有 follow-up 是挑戰用 recursion 解決，但是因為超出範圍所以也是先跳過

時間複雜度、空間複雜度分析 #

  • 時間複雜度： $O(n)$
