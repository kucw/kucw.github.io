LeetCode 刷題筆記 - 20. Valid Parentheses

古古

2024/10/30

題目 #

LeetCode 連結

給定一個 string s，裡面只會包含 '(', ')', '{', '}', '[', ']' 這些字母，檢查這個 string 是否是合法的

若是此 string 符合以下條件，即為合法：

  1. 每一個左括號都有對應的右括號
  2. 每一對左括號和右括號，他們的對應順序必須相同
  3. 每一個右括號都必須要有同樣類型的左括號

Example 1

Input: s = "()"
Output: true

Example 2

Input: s = "()[]{}"
Output: true

Example 3

Input: s = "(]"
Output: false

解答 #

程式實作 #

  • 這題是 Stack 的經典題，解法就是遇到左括號就 push 進 Stack，然後遇到右括號就 pop Stack 頂端的數據，然後比較一下這兩個是否是一對的，就能夠解決了
    • 但是有一點要注意的是，這一題很多 edge case 要考慮，譬如說在遇到右括號時，Stack 裡面是否為空（畢竟在不合法的情況中，Stack 是有可能為空的），所以針對 edge case 的考慮也是這題的重點之一
  • 程式實作

    • 這裡也用到一個小細節，就是 if not my_stack 的寫法，這個寫法等同於 Java 中的 if my_stack.isEmpty()，所以當 not my_stack 為 True 時，就是表示此 Stack 是空的（必須適應一下這個 Python 寫法…，因為好像很常用🥹）

      class Solution:
    def isValid(self, s: str) -> bool:
        my_stack = []
        pattern = {')': '(', ']': '[', '}': '{'}
        left_bracket = set(pattern.values())
        right_bracket = set(pattern.keys())

        for letter in s:
            if letter in left_bracket:
                my_stack.append(letter)

            elif letter in right_bracket:

                # 判斷一下 edge case，stack 為空是錯誤的
                if not my_stack:
                    return False

                item = my_stack.pop()

                if pattern[letter] != item:
                    return False

        # 最後也是判斷一下 edge case，stack 為空才是正確的
        if not my_stack:
            return True
        else:
            return False
