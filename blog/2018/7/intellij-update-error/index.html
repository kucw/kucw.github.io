IntelliJ - 升級遇到的問題 Error: java: invalid flag: -version

古古

2018/07/26

  • 有时候在升级 IntelliJ 時會遇到 Error: java: invalid flag: -version
  • 解決辦法
    • 打開 File -> settings，搜索 Java compiler
    • 此時會發現右下角的每一個 module 裡的 Compilation options 中，都多加了一個 -version
    • 把每個 module 的 -version 都刪除掉，就可以正常運行了
